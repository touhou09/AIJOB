# 엔드포인트: 저장소

> 최종 업데이트: {date}

---

| 저장소 | 로컬 경로 | URL | 인증 | Vault 경로 / 비고 | 상태 |
|--------|----------|-----|------|-------------------|------|
| {repo-name} | {/path/to/repo} | {github-url} | SSH / HTTPS / Token | {vault 경로 또는 접근 조건} | active |

## 상태 값
- `active`: 현재 사용 중
- `deprecated`: 사용 중단 예정
- `archived`: 보관 (읽기 전용)

## 변경 이력
- {date}: {repo} {old} → {new} (사유)
